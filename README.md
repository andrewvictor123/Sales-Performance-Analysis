# Sales Performance Analysis
A Sales Data Analysis project using SQL for cleaning and Power BI for interactive visualization.

This project focuses on transforming raw business data into actionable insights. By leveraging SQL for data cleaning and Power BI for visualization, I developed a comprehensive dashboard that tracks key performance indicators and highlights critical trends within the dataset.

Tech Stack
Database Management & Cleaning: SQL

Data Visualization: Power BI

Version Control: Git & GitHub

Workflow
Data Acquisition: Sourced the raw dataset from Kaggle.

Data Cleaning: Implemented SQL scripts to resolve data inconsistencies, handle null values, and standardize formatting.

Data Analysis: Executed SQL queries to perform exploratory data analysis and generate key metrics.

Visualization: Developed an interactive Power BI dashboard to visualize the cleaned data and monitor business performance.
